

# 📊 Sales Analysis Dashboard

## Overview
This project presents an interactive Sales Analysis Dashboard developed using Power BI. The dashboard helps analyze sales performance, profit trends, regional sales distribution, and product performance through dynamic visualizations and KPIs.

## Objectives
- Monitor overall sales and profit performance.
- Identify top-performing products and regions.
- Track sales trends over time.
- Support data-driven business decision-making.

## Dashboard Features

### 📈 Key Performance Indicators (KPIs)
- Total Sales
- Total Profit

### 📊 Visualizations
- Sales by Region
- Sales by Product
- Profit by Product
- Sales Trend Analysis
- Interactive Filters (Slicers)

### 🎯 Business Insights
- Identify high-performing regions.
- Compare product sales and profitability.
- Analyze sales patterns over time.
- Explore data interactively using slicers.

The dashboard provides a clear overview of business performance and helps stakeholders make informed decisions using sales and profit insights.
